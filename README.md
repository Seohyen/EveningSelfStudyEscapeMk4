# 야자 탈출 
#### 야간 자율 학습을 피하기 위한 선생님과의 술래잡기! 

---

###### '야자가 너무나도 하기 싫다!!' 선생님을 피해 학교를 탈출해야합니다. 교실에 있는 아이템을 사용하여 더욱 더 쉽게 선생님을 피해 달아나세요.  
###### 탈출을 위해서는 교실에 랜덤으로 숨어있는 열쇠를 찾아 교문을 열어야 합니다. 선생님에게 잡혀도 걱정 마세요!! 열심히 몸부림치며 달아날 수 있어요.
